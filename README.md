This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### 🎨 UI Components & Animations
Install UI component libraries and animation tools:
```bash
npm install @radix-ui/react-select @radix-ui/react-label @headlessui/react lucide-react framer-motion
```

### 📝 Form Handling & Validationn
For form management and schema validation:
```bash
npm install react-hook-form @hookform/resolvers zod
```

### 🎨 Tailwind CSS Setup
Set up Tailwind CSS with PostCSS and Autoprefixer:
```bash
npm install -D tailwindcss postcss autoprefixer  
npx tailwindcss init -p
```

### ✨ ESLint & Prettier (Code Formatting)
Set up ESLint and Prettier for consistent code formatting:
```bash
npm install -D eslint prettier eslint-config-prettier eslint-plugin-prettier
```
